# Screenshot Grouping Summary

This public copy does not include detailed screenshot grouping files.

The original workflow grouped rule screenshots into review buckets before creating knowledge-base entries:

- Activity and operations rules.
- Collection and progression systems.
- Card-game rules and mode variants.
- Competition and event rules.
- Card-type and priority relationships.
- Review boundaries for low-confidence OCR, shop pages, and time-sensitive event content.

Public repository boundary:

- No raw screenshots are included.
- No screenshot filenames or image IDs are included.
- No detailed reward tables, ranking pages, shop pages, or event configuration text is included.
