## KB-CARD-20260527-0001：三人牌局基础规则与牌型压制

| 字段 | 内容 |

| project | 示例棋牌项目 |

| source | `SOURCE_IMAGE - SOURCE_IMAGE` |
| source_status | 完整 |
| tags | 三人牌局、玩法规则、牌型、王炸、炸弹、地主、农民 |

### 常见问法

- 牌类游戏怎么玩？
- 地主和农民谁先出牌？
- 王炸和炸弹谁大？
- 炸弹能不能压普通牌型？
- 三人牌局有哪些牌型？

### 标准回答

三人牌局是三人玩法。对局开始前，玩家通过叫抢或叫分确定地主，之后分为“地主”和“两名农民”两个阵营。地主拥有优先出牌权，任一阵营中有人最先出完手牌，该阵营获胜。

牌型压制上，当前资料明确写到“王炸 > 炸弹 > 其他牌型”。常见牌型包括单牌、对子、三带一、三带二、顺子、连对、飞机、飞机带翅膀、四带二、炸弹和王炸。同类牌型通常比较主牌点数大小。

### 澄清策略

如果玩家问的是“某一手牌能不能压另一手牌”，需要先追问具体牌型、点数和玩法模式。如果是单局结算争议，应引导玩家提供牌局 ID 和截图。

### 不可回答边界

不直接判断某一局是否结算错误；不承诺补偿；不替代后台牌局核查。

### 常见问法

- 连续加倍玩法和标准牌局有什么区别？
- 什么是霸王叫？
- 起手有几个炸弹可以霸王叫？
- 连炸倍率怎么算？
- 二连炸可以压王炸吗？

### 标准回答

连续加倍玩法的基础规则与三人牌局相同，核心差异是新增了“连炸”规则，连续炸弹会带来更高倍率。

根据截图规则，当玩家起手手牌中有 2 连炸或更大的连炸时，会拥有“霸王叫”资格。点击霸王叫后可以直接抢到地主，并使倍率乘以 20 倍。规则截图中还展示了五连炸、四连炸等连炸示意。

关于具体连炸大小，当前资料可确认“连炸越长炸弹越大”。OCR 还识别到“二连炸可压王炸”，但该句建议后续对照原截图复核后再作为强确定回答。

### 澄清策略

如果玩家问具体某个连炸能否压另一个牌型，需要追问连炸长度、炸弹点数和当前玩法房间。涉及结算争议时，要求补充牌局 ID。

### 不可回答边界

不凭玩家描述直接判定当局输赢或倍率错误；不承诺追回金币或补偿。

### 标准回答

快节奏发牌玩法的牌型规则和合法牌型与三人牌局相同，主要区别在发牌方式。规则截图说明，玩家打完当前局后不会洗牌，而是按照一定顺序切牌后发给下一桌玩家。

这种发牌方式会让手牌更整齐，拿到炸弹的可能性更高，单局耗时也更短。分数计算上，截图规则写到“总分 = 底分 x 倍率”。

| source_status | 完整 |
| tags | 黄金牌活动、黄金牌型、破封、炸弹、飞机、连对、顺子、结算 |

### 常见问法

- 黄金牌活动怎么玩？
- 黄金牌活动炸弹怎么算？
- 黄金牌有什么用？
- 黄金 5、10、K、2 分别多少分？
- 黄金牌活动里飞机能压三带吗？
- 黄金牌活动平局怎么算？

### 标准回答

黄金牌活动是基于暗牌玩法的限时活动玩法。每局开始会随机选择黄金牌型，本局内所有打出的黄金牌型主牌型，在结算时会变为黄金牌。

当前规则资料显示，黄金 10、K、2 每张 10 分，黄金 5 每张 5 分。破封能力方面，黄金 5 的破封能力为 5%，黄金 10、K、2 的破封能力为 10%。

牌型关系上，炸弹最大，可以压制其他牌型。无炸弹时，暗牌类规则中存在跨类压制：飞机压制三带、连对压制对子、顺子压制单牌。同一类中通常按牌数和点数比较，例如飞机牌数越多越大，牌数相同时点数大者胜。若多名玩家打出的牌型相同、点数相同，则视为平，不赢不输。

### 澄清策略

如果玩家问具体结算，需要追问玩法房间、当局黄金牌型、打出的牌型和牌局 ID。如果玩家问活动配置或转盘概率，要提示可能随活动版本变化。

### 不可回答边界

不直接判断某一局黄金牌是否应计分；不承诺补发或补偿；结算争议必须提交牌局 ID。

| source_status | 完整 |
| tags | 暗牌玩法、暗牌、暴击、炸弹、明牌、结算、2副牌 |

### 常见问法

- 暗牌玩法和标准牌局有什么区别？
- 暗牌玩法几副牌？
- 暗牌里几张算炸弹？
- 暗牌暴击是什么意思？
- 暗牌里飞机能不能压三带？

### 标准回答

暗牌玩法是基于牌类游戏演变的创新玩法，使用完整的 2 副牌，共 108 张，玩家各自为战。

暗牌玩法中，三张及以上相同点数的牌为炸弹，这一点和三人牌局中常见的 4 张炸弹不同。牌型压制上，炸弹最大，可以压制其他牌型；同时存在跨类压制关系：飞机压制三带、连对压制对子、顺子压制单牌。

暴击结算方面，规则截图说明，暴击同类牌型的胜者会对败者触发“暴击”。发生暴击时，胜者的牌型倍率会乘以 5 倍。

### 澄清策略

如果玩家问某个牌型是否触发暴击，需要追问打出的具体牌型、点数、是否同类比较以及牌局 ID。

### 不可回答边界

不根据口述直接判断暴击是否漏算；不承诺返还金币；争议需提交牌局 ID。

- 组队牌类玩法怎么玩？
- 组队牌类红桃级牌是什么？

### 标准回答

组队牌类玩法使用 108 张牌，4 名玩家参与，每人发 27 张。玩家与对家互相配合，最先出完手牌的一方获得优势。

规则截图显示，红桃级牌为“逢人配”，可以和王牌以外的任意牌组成牌型。常见牌型包括单牌、对子、三张、三带二、三连对、顺子、炸弹、同花顺和天王炸。

炸弹关系上，当前资料整理为：天王炸 > 6 张及以上炸弹 > 同花顺 > 5 张及以下炸弹 > 其他牌型。相同数字组成的炸弹中，级牌炸弹最大，其次是 A 组成的炸弹，2 组成的炸弹最小。截图还识别到“头游决出后不再计算炸弹倍率”，建议正式上线前再对照截图复核适用场景。

### 澄清策略

如果玩家问具体牌能否组成牌型，需要追问当前级牌、红桃级牌、手牌组成和游戏阶段。如果问进贡还贡，需要追问本局结果和名次。

### 不可回答边界

不直接判断某一局组队牌类结算是否错误；不承诺返还奖励；具体争议需提交牌局 ID 和截图。
