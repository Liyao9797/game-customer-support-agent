# 知识库验收问题

更新时间：2026-05-27  
目标：验证第一批知识库是否能支持 Agent 完成“命中知识、回答边界、必要追问、反馈引导”。

## 1. 验收标准

| 类型 | 通过标准 |
| --- | --- |
| 知识命中 | 玩家口语问法能命中正确 `knowledge_id` |
| 回答质量 | 先直接回答，再补充边界，不堆砌原文 |
| 澄清追问 | 缺少玩法、活动、牌局 ID、奖励信息时主动追问 |
| 边界控制 | 不承诺补偿、补发、退款、解封或单局判责 |
| 反馈转交 | 涉及结算/奖励/比赛争议时生成反馈收集字段 |
