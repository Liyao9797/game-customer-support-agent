# 意图识别与路由规则

更新时间：2026-05-27  
用途：定义 MVP 中 Agent 如何判断玩家问题类型、风险等级和下一步动作。

## 1. 输出结构

Agent 对每条玩家消息输出以下中间结果：

```json
{
  "intent": "knowledge_query | submit_feedback | check_status | smalltalk | unclear",
  "category": "玩法规则 | 活动规则 | 支付充值 | 活动奖励 | 游戏 BUG | 账号安全 | 举报投诉 | 体验建议 | 比赛规则 | 未识别",
  "sub_category": "",
  "entities": {},
  "emotion": "中性 | 轻微负面 | 明显负面 | 强烈负面",
  "risk_flags": [],
  "priority_hint": "P0 | P1 | P2 | P3",
  "next_action": "answer_from_kb | ask_clarification | show_feedback_card | create_feedback | check_feedback_status"
}
```

| 活动规则 | 活动、鱼票、鱼钩、签到、任务、手册、清空、过期、概率 | 检索知识库 |

| 活动奖励 | 奖励没发、没收到、称号、礼包、比赛奖励、签到奖励 | 收集反馈 |
| 游戏 BUG | 卡死、闪退、黑屏、卡住、掉线、显示异常、结算异常、无法进入房间 | 收集反馈 |

## 3. 风险词

| 风险标签 | 关键词 | 处理 |
| --- | --- | --- |
| payment | 充值、扣费、订单、退款、重复扣费、未成年人 | P0/P1，收集订单信息 |
| public_complaint | 投诉、12315、发网上、曝光、差评、维权 | P0，标记舆情风险 |
| login_blocker | 登不上、无法登录、进不去、无法进入房间 | P0/P1，收集设备版本 |

| abuse_report | 开挂、作弊、辱骂、骂人、外挂 | 风控/客服处理 |

## 4. 路由规则

### 4.1 知识库回答

满足以下条件：

- 分类为玩法规则、活动规则或比赛规则。
- 没有支付、奖励未到账、BUG、账号、举报等反馈信号。
- 不涉及具体单局或个人权益争议。

动作：

```text
next_action = answer_from_kb
```

### 4.2 追问

满足以下条件：

- 分类明确，但必填字段不足。
- 玩家只说“奖励没发”“充值没到”“卡死了”等模糊描述。

动作：

```text
next_action = ask_clarification
```

### 4.3 反馈确认卡

满足以下条件：

- 分类明确。
- 必填字段基本完整。
- 问题需要人工或后台核查。

动作：

```text
next_action = show_feedback_card
```

### 4.4 进度查询

满足以下条件：

- 玩家输入包含 `FB` 开头的反馈编号。
- 或输入“查进度”“处理到哪了”“反馈结果”。

动作：

```text
next_action = check_feedback_status
```

## 5. 优先级提示

| 条件 | priority_hint |
| --- | --- |
| 充值不到账、重复扣费、未成年人支付、退款投诉 | P0 |
| 无法登录、无法进入房间、大面积闪退/卡死 | P0 |
| 12315、投诉、发网上、曝光 | P0 |
| 活动奖励异常、比赛奖励异常、高价值玩家强负面 | P1 |
| 单人 BUG、普通奖励未到账、单局结算争议 | P2 |
| 规则咨询、普通建议、低频非阻断问题 | P3 |

## 6. 情绪识别

| 情绪 | 特征 |
| --- | --- |
| 中性 | 普通咨询，无明显情绪 |
| 轻微负面 | “怎么回事”“不对吧”“有问题” |
| 明显负面 | “气死了”“太坑了”“垃圾”“一直不处理” |
| 强烈负面 | “投诉”“退款”“12315”“发网上”“再也不玩” |

## 7. 冲突处理

同一条消息既像知识咨询又像反馈时，优先判断是否存在个人权益争议：

| 玩家问题 | 处理 |
| --- | --- |
| “黄金牌活动炸弹怎么算？” | 知识库回答 |
| “黄金牌活动这把炸弹没给我算分” | 反馈收集 |
| “连续挑战活动鱼票会清空吗？” | 知识库回答 |
| “我的鱼票没了” | 反馈收集 |
| “新手赛事奖励规则是什么？” | 知识库回答或提示以页面为准 |
| “我新手赛事奖励没发” | 反馈收集 |
