# Public Data Notice

This repository is a sanitized public copy.

Included:

- Static demo UI for player support and operations workflows.
- Mock feedback records generated in the browser.
- An anonymized card-game knowledge-base sample.
- Smoke tests and evaluation scripts for the MVP workflow.
- Documentation describing product decisions and system boundaries.

Not included:

- Raw screenshots.
- Raw OCR CSV or Markdown exports.
- Real player data, account data, order data, customer-service records, or production work orders.
- Real API keys, webhook URLs, tenant tokens, or production integration settings.
- Real Feishu/Lark tables, documents, groups, or alert configurations.

The collaboration integration in this project is a mock adapter. It demonstrates data mapping, sync status, and retry behavior only.
